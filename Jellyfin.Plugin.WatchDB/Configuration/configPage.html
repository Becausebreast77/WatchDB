WatchDB Smart Organizer

WatchDB détecte les épisodes orphelins, confirme leur série et épisode via TMDb, puis crée une bibliothèque Jellyfin propre avec des liens symboliques. Les fichiers originaux ne sont jamais déplacés.

Dossiers d'inbox ou de releases à analyser. Les fichiers déjà rangés peuvent rester ailleurs.
Ajoute uniquement ce dossier comme source de ta bibliothèque Séries Jellyfin.
Ex. WATCHDB_TMDB_TOKEN. La variable du processus Jellyfin est prioritaire et évite d’enregistrer le secret dans la configuration du plugin.
Utilisé seulement si la variable d’environnement est absente. Ce champ reste vide après sauvegarde.
