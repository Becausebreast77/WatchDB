WatchDB Smart Organizer

WatchDB est actif automatiquement après chaque scan des bibliothèques Séries.

Il utilise les métadonnées déjà configurées dans Jellyfin, tente d’identifier les cartes inconnues, puis regroupe uniquement les séries confirmées par le même identifiant TMDb.

Aucun dossier, jeton API ou réglage à enregistrer. Tes fichiers ne sont ni déplacés ni supprimés.
